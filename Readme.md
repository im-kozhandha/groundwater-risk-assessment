# AI-Powered Groundwater Risk Assessment System

An intelligent groundwater quality risk prediction system using Machine Learning and Spatial Analytics on the CGWB 2023 dataset.

## Features
- Data preprocessing & standardization
- Groundwater contamination risk prediction
- Spatial estimation & confidence analysis
- FastAPI backend
- Streamlit frontend
- Decision Support System

## Tech Stack
- Python
- Scikit-learn
- FastAPI
- Streamlit
- Pandas
- NumPy
